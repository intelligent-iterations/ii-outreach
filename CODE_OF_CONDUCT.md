# Code Of Conduct

## Our Standard

Be respectful, direct, and constructive. Harassment, discrimination, personal attacks, and bad-faith behavior are not acceptable.

## Scope

This applies to repo discussions, issues, pull requests, and community spaces linked from this project.

## Enforcement

If you see a problem, open a private report with the maintainers instead of escalating publicly.

Maintainers may remove comments, close threads, or restrict participation when behavior makes the project harder to collaborate on safely.
